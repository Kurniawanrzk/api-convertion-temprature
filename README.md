
## API Reference

### Convert any scala

```http
  GET /api/v1/convert
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `first_scala` | `int` | **Required**. The scala before changed |
| `second_scala` | `int` | **Required**. The scala after changed  |
| `ipt_num` | `int` | **Required**. The number you want to convert  |
